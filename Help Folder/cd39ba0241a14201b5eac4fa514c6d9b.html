Create Projects

You create Build projects on the Home page of Build.

Context

You must have created a Build account and logged on to Build before you can creat a project.

Procedure

  1. Create a project by doing one of the following:
    • Click Create on the Home page if you have not previously created on become a team member of another project.
    • Click the New Project button to create a new project if you have not previously created or become a team member of another project.

    A new project tile is displayed on your Home page.

  2. Enter a name for the project, and press Enter on your keyboard.
  3. Click the project tile to display the project.

    In the Build 1.0, the Research and Feedback page is displayed. In future versions of Build, the Prototype page will be displayed when you open a project.

Project Teams

You can create a team for your project by inviting colleagues to collaborate as project team members. You can also be invited to become a team member of other Build users' projects.

Team member invited to a project can do everything in a project that Build user who created the project can, except they cannot delete or archive the project. For example, if you are invite to a project you do the following:
  • Modify the project prototype application.
  • Create user research studies and view responses to all user research studies created in the project.
  • View, add, or delete files from the project Documents folder.
  • Invite other team members to join the project.

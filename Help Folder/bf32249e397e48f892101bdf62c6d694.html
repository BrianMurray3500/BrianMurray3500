Participate in User Research Studies

To particpate in a Build user research study, participants enter a unique Build user research study-link URL in their browser and respond to questions about each image. The then enter responses to the them questions and images.

Context

To particpate in the study, participants must have received theunique Build user research study-link URL that is generated when the user study was created in the Build the study.

Procedure

  1. Click the unique Build user research study-link URL link to open it in your default browser.The home page for the study displays the study name and description.

    You can participate in Build user research studies use Google Chrome on Windows and Mac OS.

    The home page for the user research study is displayed.

  2. Read the name and the description of the study.
  3. Click the Start Study button to start responding to the study.

    The first image and question is displayed.

  4. If the study features images from an interactive HTML prototype, click the Interactive radio button on the Build title bar to make the images interactive.
  5. Click the Feedback radio button so make the image non-interactive when you want to provide feedback on the image.
  6. Click anywhere on the image to drop a Annotation icon in response to the question.

    A Comment icon is dropped on the screen, and a text entry dialog is displayed.

  7. Enter a comment if you wish, and click Done.
  8. You can drop additional Annotation icon on the image, and optionally enter a comment for each icon.
  9. When you are finished responding to this image, click the Next button to move to the next image and question.

    If there is only one question and image in the user research study, click the Done button to complete the study.

  10. When you have responded to all the questions in the study, click the Done button.

Results

Your responses are now available to the team members of the project in which the study was created.

Upload Project Documents

Files are added to the project Docs folder if you upload them directly from the Documents page, or if you add them to the project prototype application or user research studies.

Procedure

  1. One the Home page click the project tile you want to upload files to.

    In the Build 1.0, the Research and Feedback page is displayed. In future versions of Build, the Prototype page will be displayed when you open a project.

  2. Click the Docs button the navigation bar.
  3. Click Upload .

    If you have previously uploaded files to the project Docs folder, the Create link is not displayed, and you click the Upload button to upload new files.

  4. Browse to the files you want to upload, and click Open.

Results

The uploaded files are displayed on the Documents page. All documents in the projet Docs folder are displayed on the Documents page if the Everything tab is selected. You can filter the files displayed by clicking Images, or Documents, or Media tabs.
